常见问题

最后更新于2025年7月20日
识别不够准确（错别字、漏字）
+
  • 重新识别一次。文字识别的过程中使用了机器学习技术，因此每次识别的结果都会稍有不同。
  • 适当放大内容之后再进行截图。图片过小和图片质量过低是识别不准确的主要原因。
  • 在Glance的"设置-语言"中关闭自动检测语言，手动进行识别语言的设置。仅勾选所需的一种语言，或按照提示中的说明自行设定。
  • 升级macOS系统。Glance使用系统提供的文字识别框架，升级到最新的操作系统会提升识别准确度。
  • 确保图片中的文字没有过大的歪斜或遮挡等。
自动分段不够准确
+
  • 重新识别一次。文字识别的过程中使用了机器学习技术，因此每次识别的结果都会稍有不同。
  • 在Glance的"设置-通用"中调整分段灵敏度。灵敏度越低则分段中换行越少，越稳定；越高则分段中换行越多，可能会出现误判的情况。
  • 适当放大内容之后再进行截图。图片过小和图片质量过低是自动分段不准确的原因之一。
  • 截图时不要包含过多无关内容，比如空白。
  • 确保图片中的文字没有过大的歪斜或透视效果。自动分段的算法目前难以处理具有倾斜角度的文本和竖排文本。
进行文字识别时，有时候截图会反应很慢
+
  • Glance使用系统提供的截图框架，因此反应速度慢通常是系统原因，取决于当前系统的负载和调度情况。
  • 重新启动Mac可以解决这个问题。
文字识别没有反应或没有识别内容
+
  • 在"系统设置-隐私与安全性-录屏与系统录音"中检查是否授予Glance权限，如果没有请点击"+"添加Glance。
  • 如果已授予Glance权限但仍无法识别，请点击"-"移除Glance然后再次添加。
  • 确保截图包含清晰的文字内容。
  • 检查图片质量，避免模糊或过小的文字。
  • 确认选择了正确的识别语言。
发送功能无法使用
+
  • 在Glance的"设置-发送"中查看当前设定的发送链接，确保链接不为空且格式正确。
  • 确保链接指向的网站或应用程序可以正常访问。有些网站或应用程序不支持在链接中加入过长的内容。
  • 尝试更换发送链接。
按下快捷键没有反应
+
  • 在Glance的"设置-快捷键"中查看当前设定的快捷键，确保没有和常用的应用程序快捷键、系统快捷键冲突。
  • 点击相应的快捷键右边的"✕"按钮，然后重新设定快捷键。
  • 如果没有反应的是发送功能的快捷键，请先检查发送功能是否配置正常。
  • 尝试使用其他快捷键。

Frequently Asked Questions

Last updated July 20, 2025
Recognition is Not Accurate Enough
+
  • Try text recognition again. Text recognition uses machine learning technology, so results may vary slightly each time.
  • Enlarge the content appropriately before taking a screenshot. Images that are too small or of low quality are the main cause of inaccurate recognition.
  • In Glance's "Settings > Language", turn off automatic language detection and manually set the recognition language. Only check one required language or set it according to the instructions.
  • Upgrade the macOS system. Glance uses the system's text recognition framework, upgrading to the latest operating system will improve recognition accuracy.
  • Ensure that the text in the image is not severely skewed or obstructed.
Automatic Segmentation is Not Accurate Enough
+
  • Try text recognition again. Text recognition uses machine learning technology, so results may vary slightly each time.
  • Adjust the segmentation sensitivity in Glance's "Settings > General". Lower sensitivity means fewer line breaks in segments and greater stability, whereas higher sensitivity means more line breaks, which may result in misjudgements.
  • Enlarge the content appropriately before taking a screenshot. Images that are too small or of low quality can result in inaccurate automatic segmentation.
  • When taking screenshots, avoid including too much irrelevant content, such as blank spaces.
  • Ensure the text in the image isn't excessively skewed or distorted. The automatic segmentation algorithm currently has difficulty handling text with tilt angles and vertical text.
Screenshot Sometimes Responds Slowly
+
  • Glance uses the system's screenshot framework, so slow response times are usually due to system-related issues, depending on the current system load and scheduling.
  • Rebooting your Mac can solve this problem.
Text Recognition Shows No Content
+
  • Check whether Glance has been granted permission in "System Settings-Privacy and Security-Screen Recording and System Recording". If not, click "+" to add Glance.
  • If Glance has been granted permission but is still not recognized, click "-" to remove Glance and then add it again.
  • Ensure the screenshot contains clear text.
  • Check the image quality and avoid blurry or illegible text.
  • Confirm that the correct recognition language is selected.
Send Function Cannot Be Used
+
  • Check the currently set send link in Glance's "Settings > Send" to ensure the link is not empty and the format is correct.
  • Ensure that the website or application pointed to by the link can be accessed normally. Some websites or applications do not support adding overly long content in links.
  • Try changing the send link.
No Response When Pressing Shortcuts
+
  • Check the currently set shortcuts in Glance's "Settings > Shortcuts" to ensure there are no conflicts with commonly used application shortcuts or system shortcuts.
  • Click the "✕" button to the right of the corresponding shortcut, then reset the shortcut.
  • If the send function shortcut is not responding, please first check if the send function is configured properly.
  • Try using other shortcuts.
